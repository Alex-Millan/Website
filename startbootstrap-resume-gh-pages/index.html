Projects - Academics

Lock System

Hardware used: NFC, WiFi, STM32
Program language: C

Designed, budgeted and constructed a University lock system which operated at 4 AA batteries. We acheived maximum battery life of the lock by choosing a low powered microcontroller along with putting the hardware to sleep. Every 500 ms the hardware would wake up and check for any updates. Communications between the NFC and WiFi hardware was done using UART and SPI respectively. We cross reference the password entered with a database to see if the current student or faculty has access to the lock.

Date completed: June 2018

Robotic Assembly Line Simulation

Software used: V-Rep, MATLAB
Programming language: Lau

Simulated the maximum throughput of an assembly line with robots that stacked boxes. Afterwards, we disabled one of the robots and compared the new throughput. We then discovered the critical robot which slows down the production the most. All robots had an overlapping workspace so that any robot could take over the fallen robots task.

Date completed: June 2018

Autonomous Robot

Programming language: C

Built a fully autonomous vehicle with an electrical and mechanical team of 3. The robot size was 11" X 11" X 11". The robot had to navigate through a field, get ammo, shoot the ammo at a target, and repeat until the 3 minute timer ran out. An 11" x 11" x 11" obstacle was randomly place in the field which the robot had to avoid. Programmed and debugged a hierarchical state machine in an embedded system. Constructed vehicle’s frame in SolidWorks.

Date completed: December 2016

Touch That Ball

Software used: Android Studios

Created a game with 3 different colored balls bouncing across the screen. The player must tap all the balls that match the game ball in the upper-center of the screen within a time limit. The number of balls increase each level. The highest score the player reach gets recorded on their phones. Multithreading was use to allow for balls animation to occur while keeping the main GUI responsive to players.
